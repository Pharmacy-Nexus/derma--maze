# Drug dialog viewport fit fix — V4.1

The drug detail dialog now:

- fits inside the visible browser viewport at 100% zoom;
- uses a wider two-column desktop layout to reduce vertical height;
- keeps an internal scroll area only when an entry has unusually many forms or book locations;
- keeps the close button visible while the dialog content scrolls;
- switches back to a single-column, viewport-safe layout on mobile.

No drug data or treatment content was changed.
