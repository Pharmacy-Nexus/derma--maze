# Deployment

1. Delete the old repository files or create a fresh repository.
2. Upload all files and folders from this package to the repository root.
3. In GitHub: **Settings → Pages → Deploy from a branch → main → / (root)**.
4. Wait for deployment, then force-refresh the site with `Ctrl + F5`.

The public page names remain unchanged, so existing chapter URLs and canonical SEO URLs continue to work.
